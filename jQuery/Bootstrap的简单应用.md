# Bootstrap的简单应用

## 引入

参考网站：https://v3.bootcss.com/getting-started/

Bootstrap 是最受欢迎的 HTML、CSS 和 JS 框架，用于开发响应式布局、**移动设备优先**的 WEB 项目。Bootstrap是基于jQuery开发的，故而需要引入jQuery。

`````js
    <link rel="stylesheet" href="bootstrap-3.3.7-dist/css/bootstrap.min.css">
    <script src="js/jquery-3.5.1.min.js"></script>
    <script src="bootstrap-3.3.7-dist/js/bootstrap.min.js"></script>
`````

## 功能

### 栅格系统

Bootstrap 提供了一套响应式、移动设备优先的流式栅格系统，随着屏幕或视口（viewport）尺寸的增加，系统会自动分为最多**12列**。它包含了易于使用的[预定义类](https://v3.bootcss.com/css/#grid-example-basic)，还有强大的[mixin 用于生成更具语义的布局](https://v3.bootcss.com/css/#grid-less)。
